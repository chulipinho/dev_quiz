# DevQuiz

Flutter project developed during [RocketSeat's](https://www.rocketseat.com.br/) NLW 5 bootcamp.

## About

Simple quiz app.

It's developement covered topics on:
- **layout structure**
- **navigation**
- **controllers**
- **animations**
- **MaterialApp components**
- **widget creation**
- **code modularity**
- **a simple json database**
- **managing dependencies**
